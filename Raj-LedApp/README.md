# LED Pixel Map (React + Konva + TypeScript)

A browser-only pixel-mapper for LED walls (snapping, groups, background & logo, export PNG/PDF).

## Quick start
```bash
npm install
npm run dev
```

Open http://localhost:5173 then in the **Export** panel click **Load 3×3 Demo**.

## Controls
- Wheel: zoom (focused at cursor)
- Space + drag or Middle mouse: pan
- Shift-click: multi-select
- Cmd/Ctrl + D: duplicate
- Del/Backspace: delete
- Arrow keys: nudge (Shift ×10)
- Export: PNG or PDF

## Notes
- Client-side only, autosaves to localStorage.
- Use "Download JSON" to export your project state.
